[LeetCode]907-Sum of SubArray Minimum

Posted on 2019-08-30 | In Leetcode Solution |

This is a typical problem to use a data structure called Monotone Stack
A Monotone Stack is a stack with its element in increasing or decreasing order

Problem Description

Given an array of integers A, find the sum of min(B), where B ranges over every (contiguous) subarray > of A.Since the answer may be large, return the answer modulo 10^9 + 7.

Solution Code

Thinking Behind Code

First, it’s easy to come up with and easy to rule out the brute force solution with O(N^2) time complexity. Then we shall take a step further to come up with the idea that this problem can be simplified as a problem to find the index of first left element and first right element that are less than the given element. Because between the two indexs, all the subarrays containing the given element will return a maximum element of the given element, because it’s the smallest. Therefore the result is equal to the given element A[i]*(#subarray).

So it all boils down to find the left and right index. Here comes our monotone stack.Taking the left side as an example. I will clarify some key code.

To initialize the stack, the index of first smaller element should be set to -1 which means there is an imaginery element indexed -1 to calculate the distance.

If current element is greater than the top leftStack, the first left smaller element is leftStack.top().
In other case, pop the top out of stack until is empty or find the smaller one.
Pay attention to >= symbol You should use unstrict greater than(>=) to make sure to handle the equal elements properly. You can use >= once, either in left loop or right loop.
